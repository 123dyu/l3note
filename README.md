# l3note

一份 `LaTeX3` 学习笔记，希望这份笔记（文档）能够帮到对 `LaTeX3` 感兴趣的人。

## PDF文档获取方法
- 获取整份PDF文档：用命令 `xelatex `编译 `l3note.tex` 文件
  ```
  xelatex yi.dtx
  ```
- 单独获取各模块的PDF文档：用命令 `xelatex `编译 `sourch` 文件夹下的`*-cn.dtx`文件，比如：
  ```
  xelatex l3names-cn.dtx
  ```
